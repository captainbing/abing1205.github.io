python基本语法

python是一种解释性的脚本语言,解释一行执行一行

依靠缩进来区分代码块

变量

  • <% label del red%>在内存中删除变量
  • <% label dir()函数 green%>可以查看当前变量可以调用那些函数
  • <% label eval()函数 orange%>可以将字符串当作函数表达式使用
  • 全局变量 ps: <% label global pink%>关键字，在函数内部使用外部变量
  • 局部变量 定义在函数内部中的变量
  • <% label // green%> 运算符代表整除 <% label ** green%>代表幂运算

基本数据结构

  1. 整型 int

  2. 浮点型 float

  3. 布尔型 bool True False

  4. 字符型 str name=”你好呀世界”

    str()转化为字符串函数

    name[3] 取出第四个字符

  5. 列表 list list() 数组 []

    extends() 拼接值 += 拼接列表一致

    append() 直接拼接列表对象，而不是值

  6. 元组 与列表一致,只是元素不可改变

    tuple()将其他类型转化成元组

  1. 字典 dict <% label 键值对 pink%>

    类似于json

    person = { “name”: “小明” }

    person[“name”] 取出字典中的元素

逻辑运算符

  • and 与 类似 &&
  • or 或 类似 |
  • not 非 类似 ！

身份运算符

  • <% label is green%> 和 is not 判断引用对象是否是同一个
  • ==是判断引用对象的值是否相等

流程控制语句

1.条件控制语句

2.循环控制语句

函数

  • 函数定义关键字 def

面向对象

  • 对象名一般采用大驼峰法
  • __ 双下划线代表私有
  • mro()方法可以查看方法解析顺序
  • 装饰器
    1. @classmethod 类方法
    2. @staticmethod 静态方法
  • 面向对象三大特性：
  1. 封装
  2. 继承
    • 单继承
    • 多继承：可以继承多个父类
    • 私有属性和私有方法不被继承
  3. 多态

异常

  • raise： 手动抛异常
  • except：发生异常时执行
  • else： 未发生异常时执行
  • finally： 一定会执行

模块

  1. import 模块名 as 别名 直接导入模块
  2. from ··· import 导入模块中指定内容
